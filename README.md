# Классификация новостей Банка России для прогнозирования изменений ключевой ставки

## Описание проекта

Проект направлен на создание модели машинного обучения, которая по текстам новостей и официальных пресс-релизов Банка России прогнозирует ближайшее решение по ключевой ставке. Модель классифицирует текст на три категории:

- `down` - снижение ставки
- `same` - ставка без изменений
- `up` - повышение ставки

## Технические детали

### Структура проекта

cbr-news-prediction/
├── configs/ # Конфигурационные файлы Hydra
├── cbr_news/ # Исходный код Python
├── data/ # Данные (управляются через DVC)
├── outputs/ # Результаты тренировок
├── checkpoints/ # Сохраненные модели
└── tests/ # Тесты

### Требования

- Python 3.10+
- PyTorch 2.0+
- CUDA (рекомендуется для тренировки)

## Установка и настройка

### 1. Клонирование репозитория

```bash
git clone https://github.com/yourusername/cbr-news-prediction.git
cd cbr-news-prediction
```

### 2. Создание виртуального окружения

poetry install
poetry shell

### 3. Настройка pre-commit

pre-commit install
pre-commit run -a # Проверка всех файлов

### 4. Настройка DVC (опционально)

dvc init
dvc remote add -d myremote gdrive://your-folder-id

Запуск тренировки
python train.py
